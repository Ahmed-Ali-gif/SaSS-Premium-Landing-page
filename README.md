# 🚀 ApexFlow - Modern SaaS Landing Page

A modern, responsive SaaS landing page built with HTML and CSS. This project focuses on clean UI, accessibility, responsive layouts, and modern design principles inspired by real-world SaaS websites.

## 📸 Preview

> A clean business landing page featuring:
- Hero section
- Dashboard mockup
- Trusted companies
- Features section
- Testimonials
- Pricing plans
- FAQ accordion
- Call-to-Action section
- Professional footer
